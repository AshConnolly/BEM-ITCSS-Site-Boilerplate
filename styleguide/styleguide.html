All column elements are 100% width by default.

Full width is default - this causes the grid to expand so that the content of the gird is inline with its parents content: like this paragraph.

Note the class 'sg-col' is used in this styleguide to better present the columns, and is not needed in practice.

Columns are split into 100%, 75%, 66%, 50%, 40%, 33%, 25% widths.

Responsive changes are made by using classes: '.l-col-m-50' will cause a column to change to 50% width at the medium breakpoint and upwards (mobile first).
